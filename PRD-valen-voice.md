# PRD — 발렌 어투 개편: 거만한 황제 사극체

## 배경
사용자 요청(2026-07): 발렌의 대사를 "~이느니라" 같은 거만·오만한 말투로 전면 전환(본편 포함).

## 어투 규칙 (SSOT — 이후 발렌 신규 대사는 이 규칙을 따른다)
- **1인칭**: 짐 (기존 유지)
- **평서 종결**: ~이니라 / ~느니라 / ~노라 / ~것이니라 / ~ㄹ 터 / ~하느니라
- **의문**: ~느냐 / ~인가 / ~겠는가 / 어이하여 ~ㄴ고
- **명령/청유**: ~하라 / ~할지어다
- **상대 지칭**: 그대(하대), 늙은이·개 등 오만한 비칭 유지
- **감정 붕괴 비트**(vep4~5 sad/distressed): 형식은 사극체 유지하되 탄식 어미(~구나/~노라)로 인간미 노출 — 오만함이 갈라지는 연출
- **금지**: ~해/~야/~거든/~네 같은 부드러운 현대 반말 종결
- **정본 대사 보존**: "재상, 짐의 검을 시험하는가"(ep9) 계열은 유지, '축하→경하'만 통일
